CSS Transitions and Animations Flashcards

What is transition?
Makes a property change smoothly over time.
How do you add a transition?
transition: 0.5s;
What is animation?
Creates movement using keyframes.
What is @keyframes?
Defines the steps of an animation.
animation-duration
Sets how long the animation lasts.
animation-name
Chooses which keyframes to use.
:hover
Applies styles when the mouse is over an element.
opacity
Controls transparency from 0 to 1.
